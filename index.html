Documentação do Desenvolvimento

Descrição Geral do Projeto

Este projeto consiste em um site de portfólio pessoal simples, desenvolvido com HTML, CSS e JavaScript. O objetivo principal foi criar uma plataforma para apresentar minhas habilidades, projetos e formas de contato de maneira visualmente agradável e funcional.

Processo de Desenvolvimento

O desenvolvimento foi dividido em etapas bem definidas para facilitar o gerenciamento e a organização do projeto.

1. Planejamento

Durante o planejamento, defini os objetivos, funcionalidades e um cronograma inicial:

  • Estruturar as seções do site (Sobre, Habilidades, Projetos, Contato).
  • Estilizar o site com CSS e torná-lo responsivo.
  • Publicar o site no GitHub Pages.

2. Desenvolvimento por Etapas

  • Criação do layout em HTML.
  • Estilização com CSS para um design limpo e profissional.
  • Adição de interatividade básica com JavaScript.

3. Uso do Git e Benefícios

O Git foi essencial para gerenciar o progresso do projeto. Aqui estão os principais desafios e como o Git ajudou:

  • Gerenciamento de funcionalidades: Usei branches separadas para trabalhar em diferentes partes do site.
  • Correção de erros: Commits frequentes permitiram identificar rapidamente bugs no código.
  • Publicação online: O GitHub Pages foi usado para compartilhar o projeto final.

Conclusão

Este projeto foi um exercício valioso para consolidar práticas de desenvolvimento web e uso de Git. Os desafios enfrentados tornaram o aprendizado mais significativo.
